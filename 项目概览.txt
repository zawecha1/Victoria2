Victoria II Save Game Parser - 项目文件概览
================================================

📁 项目文件结构:
├── China2245_04_06.v2           # Victoria II存档文件（示例）
├── victoria2_parser.py          # 🔧 完整解析器 - 详细解析所有数据
├── simple_parser.py             # ⚡ 简化解析器 - 快速提取关键信息  
├── analyze_data.py              # 📊 数据分析工具 - 深度分析报告
├── test_parser.py               # 🧪 测试脚本 - 验证解析功能
├── run_parser.bat               # 🖱️ Windows批处理 - 一键运行
├── README.md                    # 📖 详细使用说明
└── 项目概览.txt                 # 📋 本文件

🚀 快速开始:
1. 双击 run_parser.bat (Windows)
2. 或运行: python simple_parser.py
3. 分析数据: python analyze_data.py

📊 生成的文件:
├── victoria2_analysis.json      # 📄 基础分析结果
├── china_save_parsed.json       # 📄 完整解析数据
└── *_analysis_report.txt        # 📄 深度分析报告

🎯 主要功能:
• 解析Victoria II .v2存档文件
• 提取国家、省份、世界市场数据
• 生成经济排名和统计分析
• 导出JSON格式的结构化数据
• 生成可读的分析报告

💡 推荐使用流程:
1. 运行 simple_parser.py 快速分析
2. 运行 analyze_data.py 查看详细报告
3. 如需完整数据，运行 victoria2_parser.py

🔧 技术特点:
• 支持大型存档文件（100MB+）
• 智能数据类型转换
• 递归解析嵌套结构
• 错误处理和容错机制
• 多种输出格式
